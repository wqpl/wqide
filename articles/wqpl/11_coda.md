# Coda

Congratulations! You have now seen the core ideas behind wq:

- Lists as a central data type
- Dicts for named structure
- Everything is an expression
- Values as functions
- Data flows naturally via broadcasting and pipelines
- Concise functions

## Mindset

wq is designed to encourage:

- Thinking in expressions
- Trying list operations before writing manual loops
- Using short functions freely
- Building understanding by running small variations of examples
